📖 操作說明

基本操作

  • 從左側「元件庫」點選元件加入畫布,拖曳元件本體可移動位置。
  • 上方選擇線色:紅線=火線、白線=中性線。
  • 拉線:按住任一端子(圓點)拖曳到另一個端子放開,即完成接線。
  • 點選導線或元件後按 Delete 刪除;點選導線也會出現 ✕ 可刪除。
  • 調整導線弧度:點選導線後拖曳中間的圓形把手(或直接按住線身拖曳),把線彎到不遮擋的位置;雙擊導線可恢復預設曲線。
  • 點擊 NFB 把手、開關搖板可切換 ON / OFF。
  • Ctrl+Z 復原、Ctrl+Y 重做,步數不限;接線進度會自動儲存在瀏覽器,下次開啟自動還原。
  • 「插座+燈泡」= 燈泡直接插在插座上:插座接線正確通電時,中間的燈泡直接亮起,不必再拉線接燈泡。

範例一:單切開關控一盞燈

  1. 電源 L(火線)→ 紅線 → NFB 電源側 L;電源 N → 白線 → NFB 電源側 N。
  2. NFB 負載側 L → 紅線 → 單切開關上方任一孔。
  3. 開關下方任一孔 → 紅線 → 燈泡一端(上下兩孔各為一極,同排兩孔互通)。
  4. 燈泡另一端 → 白線 → NFB 負載側 N。
  5. 打開 NFB、打開開關 → 燈亮,導線上出現電流流動動畫!

範例二:兩顆雙切開關控一燈(樓梯燈)

  1. 加入兩顆雙切開關(A、B)。上方為共用端 0,下方為切換端 1、3。
  2. 火線 → A 的共用端 0。
  3. A 的 1 ↔ B 的 1、A 的 3 ↔ B 的 3(兩條紅線)。
  4. B 的共用端 0 → 燈泡 → 白線回中性線。
  5. 完成後,切換任何一顆開關都能開/關燈。

⚠ 注意

  • 火線若不經負載直接接回中性線會造成短路,送電時 NFB 會跳脫!
  • NFB 跳脫後,需先把把手切到 OFF、排除短路,才能重新送電。
  • 插座背面:銅色螺絲=火線側(短孔)、銀色螺絲=中性線側(長孔)。
