# ⚡ 配電練習小工具 (DistroCraft)

一個互動式的網頁版基礎配電模擬練習工具，讓使用者可以透過拖曳、接線來模擬與學習基本的室內配電原理。

## 🌟 功能特色

* **互動式接線**：提供火線（紅）與中性線（白）兩種導線顏色，點擊元件端子即可拉線，並可自由調整導線弧度。
* **豐富元件庫**：包含電源 (AC 110V)、NFB 無熔絲斷路器、端子台、單切開關、雙切開關（三路開關）、插座及燈泡等。
* **即時通電模擬**：接線正確時，切換開關或 NFB 把手，導線上會即時顯示電流流動的動畫，燈泡會發亮。
* **短路偵測保護**：若發生火線與中性線直接短路，系統會立即提示，並模擬 NFB 斷路器「跳脫 (Tripped)」，必須排除短路並重置 NFB 才能重新送電。
* **內建範例模板**：一鍵載入經典的「雙切開關控制一燈（樓梯燈）」範例，方便初學者觀摩學習。
* **進度自動儲存**：接線進度與畫面狀態會自動儲存在瀏覽器中，重新整理或下次開啟時不會遺失進度，並支援無限制的復原與重做 (Ctrl+Z / Ctrl+Y)。

## 📖 操作說明

* **加入元件**：點擊左側「元件庫」按鈕。
* **移動元件**：拖曳畫布上的元件本體。
* **拉線**：按住元件的端子（圓點）拖曳到另一個端子放開。
* **刪除**：點選導線或元件後按 `Delete` 鍵。
* **調整視角**：拖曳空白處可平移畫布，滾輪可縮放檢視比例。
